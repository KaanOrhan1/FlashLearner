# FlashLearner

FlashLearner is a web-based learning application designed to help users remember information more effectively using spaced repetition.

The system automatically schedules flashcard reviews based on performance, making studying more efficient and personalised.

This project was developed as part of my Final Year Project in Computer Science.

---

## Live Demo
https://flashlearn-49cc8.web.app

---

## Features

- Create and manage flashcards
- AI-assisted flashcard generation
- Personalised review schedule using the SM-2 algorithm
- Track learning progress
- Simple and distraction-free interface
- User authentication and cloud storage

---

## Tech Stack

- React
- Firebase (Authentication, Firestore, Hosting)
- JavaScript
- HTML / CSS

---

## About the Project

The goal of this project was to explore how algorithm-based learning techniques can improve long-term memory retention.

FlashLearner uses the SM-2 spaced repetition algorithm to determine the optimal time to review each flashcard, helping users focus on the material they are most likely to forget.

The application was designed with simplicity in mind, allowing users to focus on learning without unnecessary distractions.
